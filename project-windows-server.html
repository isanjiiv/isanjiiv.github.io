Windows Server Domain Setup + GPO Automation + Backup

Windows Server 2016 AD DS & DNS GPO Acronis Cyber Protect

Overview

Designed and implemented a complete Windows Server 2016 domain infrastructure for an enterprise environment to centralize user management, enforce security policies, automate IT operations, and ensure reliable data protection. This project included Active Directory Domain Services (AD DS), DNS integration, structured OU design, Group Policy management, shared folder access control, printer deployment, and a fully scheduled Acronis Cyber Protect Cloud backup strategy.

Key Implementation Details

1) Active Directory Domain Services (AD DS)

Configured and managed a Windows Server 2016 domain environment to support centralized authentication and access control.

  • Installed and configured AD DS and DNS roles.
  • Created a domain structure for enterprise-level user/device management.
  • Managed domain objects through Active Directory Users and Computers (ADUC).

2) Department-Wise OU Structure

Built a department-based Organizational Unit (OU) hierarchy (Accounts, HR, IT, Design, etc.) to simplify administration and enable targeted policy application.

3) Group Policy (GPO) Implementation

Implemented multiple GPOs to automate workstation configuration and enforce security.

  • USB Access Deny: Restricted USB storage usage to prevent unauthorized data transfer.
  • Drive Mapping: Automatically mapped shared drives for teams like Design/IT.
  • Printer Deployment: Automated printer deployment without manual installation.

4) Backup & Disaster Recovery (Acronis)

Implemented a secure backup strategy using Acronis Cyber Protect Cloud. Configured daily incremental backups for critical systems and shared folders to ensure fast recovery.
