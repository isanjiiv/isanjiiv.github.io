Windows Server Domain Setup + GPO Automation + Backup & Recovery

Windows Server 2016 Active Directory (AD DS) GPO Acronis Cyber Protect

Overview

Designed and implemented a complete Windows Server 2016 domain infrastructure for an enterprise environment to centralize user management, enforce security policies, automate IT operations, and ensure reliable data protection. This project included Active Directory Domain Services (AD DS), DNS integration, structured OU design, Group Policy management, shared folder access control, printer deployment, and a fully scheduled Acronis Cyber Protect Cloud backup strategy for disaster recovery readiness.

Key Responsibilities & Implementation

1) Active Directory Domain Services (AD DS) Deployment

Configured and managed a Windows Server 2016 domain environment to support centralized authentication and access control across the organization.

  • Installed and configured AD DS and DNS roles
  • Created a domain structure for enterprise-level user and device management
  • Managed domain objects through Active Directory Users and Computers (ADUC)

2) Department-Wise OU Structure & User Management

Built a department-based Organizational Unit (OU) hierarchy to simplify administration and enable targeted policy application.

  • Created OUs for multiple teams: Accounts, Costing, CRM, Design, HR, IT, Purchase, Reception, and Management.
  • Added and managed user accounts, security groups, and role-based access.

3) Group Policy (GPO) Implementation

Implemented multiple Group Policy Objects (GPOs) to automate workstation configuration and improve security.

  • USB Access Deny Policy: Restricted USB storage usage.
  • Drive Mapping Policy: Automatically mapped shared drives for teams like Design/IT.
  • Printer Deployment Policy: Automated printer deployment without manual installation.

4) Backup & Disaster Recovery using Acronis

Implemented a secure backup strategy using Acronis Cyber Protect Cloud to protect critical user data and business folders.

  • Configured backup plans for important systems and shared folders.
  • Set up scheduled daily incremental backups.

Fortinet 60F SD-WAN Deployment

FortiGate 60F SD-WAN VLAN

Overview

Configured and deployed a FortiGate 60F firewall to secure the company network and improve internet reliability. The key focus was implementing SD-WAN to manage two ISP connections (Jio & Spectra) for redundancy and load balancing.

Key Implementation Details

1) SD-WAN Configuration

Integrated multiple ISP links (Jio Fiber and Spectra) into a single logical interface using SD-WAN. Configured rules to route business-critical traffic through the most stable line and enabled automatic failover.

2) Security Policies

Applied strict firewall policies to control traffic flow between internal VLANs and the WAN. Enabled Web Filtering and Application Control to block non-business websites and risky applications.

Secure Wifi Network (FortiGate + Deco Mesh)

Deco Mesh Fortinet WiFi Security

Overview

Implemented a secure wireless network upgrade using TP-Link Deco Mesh systems integrated with the existing FortiGate 60F firewall. The goal was to provide seamless roaming coverage while ensuring strong security.

Key Implementation Details

1) Mesh Network Setup

Deployed multiple Deco Mesh units in Access Point mode to cover dead zones in the office. Configured seamless roaming.

2) Integration with FortiGate

Connected the Mesh system to a dedicated interface on the FortiGate 60F. This allows the firewall to inspect all WiFi traffic.
